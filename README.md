
# Margin Maestro Pro

An industrial-styled gross margin calculation tool for streetwear brands and retailers.

## Features
- **Real-time Cost Analysis**: Calculate landed costs including shipping and taxes.
- **Visual Feedback**: Industrial brutalist design with dynamic color-coded margins.
- **Scenario Simulation**: Instant profit checks for 30%, 50%, and 70% off discount scenarios.
- **Responsive Design**: Optimized for both desktop and mobile operations.

## Tech Stack
- React 18+ (Vite)
- TypeScript
- Tailwind CSS (via CDN for hybrid portability)

## How to Run (Local Development)

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
